# Test project
Project for testing and learning <b>JavaScript/TypeScript</b>, <b>React</b>, <b>Python</b>, <b>databases</b>, <b>HTML</b>, <b>CSS</b>.  

I´m using here database on localhost, MySQL service.  

It is kind of event management application divided to more parts.
1. Searching for events and their details
2. Register to an event
3. Editing profile
4. Admin features like creating new event, updating and deleting. 

## Current state
### What is working fine:
1. User Authentication (Context,cookies)
2. Data fetching, filtering based on some criteria (Axios)
3. Routing, subpages (React Router)
4. Carousel/slider :)
5. New user registration
